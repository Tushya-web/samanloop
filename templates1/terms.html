Acceptance

By interacting with SamanLoop, you acknowledge that you have read and understood these policies. This platform is built on community trust and academic integrity.

Academic Disclaimer

SamanLoop is an academic project developed strictly for educational and demonstration purposes.

  • This is a non-commercial prototype.
  • No actual legal or financial guarantees are provided by the developers.
  • Features are designed to showcase Full-Stack System Design concepts.
All data and transactions are simulated. Users should not use real sensitive financial data.

User Identity & Ethics

To ensure a high-quality simulation environment:

  • Users must be 18 years or older.
  • Academic honesty is expected; do not create multiple "test" accounts for malicious purposes.
  • Identity verification is a simulated flow to demonstrate security logic.

Truthfulness & Anti-Fraud

Even in a demo, we enforce a strict honesty policy:

  • Fake listings or intentionally misleading descriptions are prohibited.
  • Users found "gaming" the simulated rating system will have their data reset.
  • Any attempt to bypass system logic for simulated gains is restricted.

Role Responsibilities

Lenders: Must provide accurate item condition and clear safety instructions.

Borrowers: Responsible for returning items on time and in the condition received.

Limitation of Liability

Disclaimer: The developers assume no responsibility for disputes between users or any real-world actions resulting from this academic demonstration.

Payment Simulation

The "Wallet" and "Escrow" systems are for demonstration logic only.

  • No real currency is exchanged or stored.
  • Mocked payment gateways are used for the transaction flow.
  • Any displayed currency is virtual "demo credits."

Data & Privacy

We treat simulated data with care. User profiles are encrypted using standard hashing (PBKDF2). Data is stored solely for project evaluation and will be purged upon completion of the academic term.
